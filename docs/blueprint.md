# **App Name**: TrackRuta

## Core Features:

- Morning Shift Check-In: Records the route number, driver ID, license plate, date, current timestamp (for check-in), and any observations.
- Morning Shift Check-Out: Records the timestamp for when the route is finished in the morning shift and enables a signature button.
- Afternoon Shift Check-In: Records the timestamp for the beginning of the afternoon shift, alongside route number, driver ID, license plate, date, and observations.
- Afternoon Shift Check-Out: Records the timestamp for the route's completion in the afternoon shift and enables the signature button.
- Digital Signature Capture: Allows users to input their signature or name via a digital input method when checking in and out for audit compliance
- Route Data Storage: Stores all recorded route data, including timestamps, driver information, and signatures, in a structured database.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to convey professionalism and trust.
- Background color: Very light grey (#F0F2F5) for a clean, readable interface.
- Accent color: Electric purple (#8a2be2) for focus and to guide user interaction.
- Body and headline font: 'Inter' for a modern and easily readable user experience.
- Use simple, clear icons for navigation and status indicators.
- Design a layout optimized for quick data entry and review on mobile devices.
- Use subtle transitions and feedback animations to confirm actions.
